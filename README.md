# Speech Transcription App
A web-based speech-to-text application using Microsoft Azure Speech Services with support for multiple languages.

## Features
Real-time Transcription: Convert speech to text in real-time

Multi-language Support: English, French, Simplified Chinese, and Traditional Chinese

Easy Export: Download transcripts as text files

Beautiful Interface: Temple-themed design with clean, readable layout

Configuration Persistence: Saves your Azure settings automatically

## Setup Instructions
### 1. Get Azure Credentials:

Sign up for Azure Cognitive Services

Create a Speech Service resource

Copy your Subscription Key and Region

### 2. Configure the App:

Enter your Azure Subscription Key

Select your Service Region (e.g., "eastus")

Click "Save Configuration"

### 3. Start Transcribing:

Select your language

Click "Start Listening"

Allow microphone access when prompted

Begin speaking

## Usage
Start/Stop: Use the green and red buttons to control recording

Clear: Remove current transcript

Download: Save transcript as a text file

Language: Switch between supported languages

## Supported Languages
English (United States) - en-US

French (France) - fr-FR

Chinese Simplified - zh-CN

Chinese Traditional - zh-TW

## Technical Requirements
Modern web browser with microphone access

Microsoft Azure Speech Service subscription

Internet connection

## Privacy
All speech processing happens through Azure's secure services

No audio data is stored on our servers

Configuration is saved locally in your browser
